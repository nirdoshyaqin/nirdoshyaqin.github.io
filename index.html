Web Designer Yuya Sato

東京都新宿区在住、週末湘南藤沢、ときどき信州で活動しているWebデザイナー、佐藤　裕也です。WordPressを使って制作したチベット・ネパール曼荼羅専門ショップというお店を主にオンライベースで企画・運営しております。 ネパールという国から曼荼羅（マンダラ）という仏画、スピリチュアルアートを仕入れて、販売しております。

プログラミング言語、デザインスキルを習得することで収入の入り口をもう一つ増やしたい思い、 TechAcademyにてWordpress × Web Designのコースを思い切って受講。このコースを通して、WordPress × Web Design × PHPの基礎土台を固めて主にクラウドワークスやランサーズなどの外注サイトを通して月に10万円の収入を安定的に継続的に得ることが直近の目標です。
